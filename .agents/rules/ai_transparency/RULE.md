# AI Transparency Rule

This rule defines how AI-generated financial content must be presented.

## Standards
- **Disclaimer**: All financial outputs **MUST** include: *"This is a student hackathon project. Outputs are not investment advice."*
- **Uncertainty**: Label all forecasted metrics clearly (e.g., "Estimated", "Forecasted").
- **Scenarios**: For the financial model, explicitly define the inputs and assumptions for Base, Upside, and Downside cases.
- **Reasoning**: Log the multi-step reasoning (traces) to ensure observable agentic flows.
